# WeChat-Kimi Bridge - 立即使用指南

## 🚀 最简单的方法（推荐）

### 第一步：双击运行
找到文件夹中的 `一键启动.bat`，**双击运行**

### 第二步：立即扫码
- 程序会显示二维码链接
- 立即用手机微信扫描
- 点击"确认登录"

### 第三步：开始使用
登录成功后就可以：
- 私聊给机器人发消息
- 群聊中 @Kimi 发送消息

---

## 📱 如果二维码过期怎么办？

### 方法1：重新运行
关闭程序，重新双击 `一键启动.bat`，获取新二维码

### 方法2：命令行运行（更快）
1. 打开命令行（Win+R，输入cmd）
2. 进入项目目录：
   ```bash
   cd C:\Users\lenovo\wechat-kimi-project
   ```
3. 运行程序：
   ```bash
   python wechat-kimi-bridge-stable.py
   ```
4. **立即扫码**（5分钟内有效）

## ❗ 常见问题

### Q: 二维码总是过期？
A: 二维码有效期约5分钟，请：
- 提前打开微信扫一扫
- 程序一运行就立即扫码
- 如果过期，关闭重新运行

### Q: 扫描后没有反应？
A: 确保：
- 点击了"确认登录"
- 手机网络正常
- 等待几秒钟

### Q: 登录失败？
A: 尝试：
1. 关闭防火墙
2. 更换网络环境
3. 重新运行程序
