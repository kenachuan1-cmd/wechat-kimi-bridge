## 🚀 推荐方案：GitHub Codespaces（最佳）

### 为什么推荐？

✅ **网络最稳定** - GitHub服务器直连微信
✅ **环境已配置** - 自动安装所有依赖
✅ **随时随地访问** - 任何设备都能用
✅ **二维码实时显示** - 无延迟问题

### 操作步骤（5分钟）

#### 第1步：上传到GitHub（2分钟）

**方法一：网页上传（最简单）**
1. 访问 https://github.com/new
2. 创建新仓库
3. 点击 "uploading an existing file"
4. 上传项目文件夹中的所有文件

**方法二：命令行**
```bash
cd C:\Users\lenovo\wechat-kimi-project
git init
git add .
git commit -m "Initial commit"
# 然后按提示推送到GitHub
```

#### 第2步：打开Codespaces（1分钟）

1. 访问你的GitHub仓库页面
2. 点击绿色 **"<> Code"** 按钮
3. 选择 **"Codespaces"** 标签
4. 点击 **"Create codespace on main"**

#### 第3步：运行程序（2分钟）

1. 等待环境准备好（约1分钟）
2. 在终端输入：
   ```bash
   python wechat-kimi-bridge-stable.py
   ```
3. **立即扫码**（二维码显示在终端中）
4. 完成！

## 🏠 备选方案：本地运行

### 适合：不想用GitHub的用户

**步骤：**
1. 打开命令行（Win+R → cmd）
2. 进入项目目录：
   ```bash
   cd C:\Users\lenovo\wechat-kimi-project
   ```
3. 运行程序：
   ```bash
   python wechat-kimi-bridge-stable.py
   ```
4. **立即扫码**（二维码显示在命令行中）

**注意：**
- 需要提前打开微信扫一扫

- 如果过期，关闭重新运行
